🐾 汪洁步道——智能宠物清洗神器

📖 项目简介
汪洁步道是一款基于AI情绪识别的智能宠物清洗设备，专为解决遛狗回家后的"泥爪灾难"而设计。通过计算机视觉实时监测狗狗情绪，自适应调节清洗刷转速，让宠物清洗变得轻松、智能、零压力。

✨ 核心功能
🐕 智能情绪识别模块
实时情绪分析: 使用GLM-4V视觉模型实时识别8种狗狗情绪状态

自适应控制: 根据情绪动态调整清洗刷转速（20-80rpm可调）

平滑过渡: 模拟真实电机加减速，避免突然变化刺激宠物

可视化反馈: 实时显示情绪分布与清洗状态

🔧 技术特性
前端: HTML5 + JavaScript + Canvas + Chart.js

AI模型: 智谱GLM-4V视觉大模型

数据格式: 标准化JSON输出，便于硬件集成

配置管理: 本地化API Key存储与配置

🎯 解决的问题
🌧️ 传统洗狗痛点
主人累: 弯腰擦脚到"质壁分离"

狗狗烦: 抗拒清洗，关系紧张

卫生差: 泥水带进屋内，清洁困难

费用高: 宠物店清洗昂贵且耗时

💡 我们的解决方案
智能识别: AI情绪感知，读懂狗狗状态

自适应清洗: 紧张时减速，放松时加速

一键操作: 狗狗进入即自动清洗

保护关系: 减少对抗，提升互动质量

📊 情绪识别能力
系统可识别8种狗狗常见情绪：

情绪类别	描述	典型行为表现
😊 快乐兴奋	尾巴高摇，身体放松	主动接近，眼神明亮
🧘 放松平静	身体舒展，呼吸均匀	安静站立，享受清洗
👀 好奇警觉	耳朵竖起，头部前倾	对环境敏感，但无恐惧
😰 焦虑紧张	身体紧绷，尾巴低垂	试图逃离，眼神躲避
🎮 顽皮爱玩	跳跃，轻咬	互动性强，精力充沛
🔍 专注投入	目光锁定，身体稳定	对清洗过程感兴趣
😴 疲倦困乏	动作缓慢，眼皮下垂	配合但无精打采
🤝 友好亲近	身体靠近，舔舐动作	信任主人，享受互动
⚙️ 情绪-转速映射逻辑
javascript
// 核心控制逻辑
if (主情绪 == "焦虑紧张" || "好奇警觉" || 紧张情绪占比 > 30%) {
    targetSpeed = 20;  // 安全低速，安抚情绪
} else if (主情绪 == "放松平静" || "友好亲近" || 放松情绪占比 > 60%) {
    targetSpeed = 75;  // 高效清洁，舒适速度
} else {
    targetSpeed = 60;  // 默认舒适速度
}
